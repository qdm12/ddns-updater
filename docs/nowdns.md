# Now-DNS

## Configuration

### Example

```json
{
  "settings": [
    {
      "provider": "nowdns",
      "domain": "domain.com",
      "username": "username",
      "password": "password",
    }
  ]
}
```

### Compulsory parameters

- `"domain"` your full domain name (FQDN)
- `"username"` your email address
- `"password"`
