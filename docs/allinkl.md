# All-Inkl

## Configuration

### Example

```json
{
  "settings": [
    {
      "provider": "allinkl",
      "domain": "domain.com",
      "host": "host",
      "username": "dynXXXXXXX",
      "password": "password"
    }
  ]
}
```

### Compulsory parameters

- `"domain"`
- `"host"` is your host (subdomain)
- `"username"` username (usually starts with dyn followed by numbers)
- `"password"` password in plain text
